### **React 프로젝트 폴더 용도**
1. **`components/`** - 재사용 가능한 UI 컴포넌트. (예: 버튼, 모달)
2. **`pages/`** - 각 페이지 단위 컴포넌트. (예: 홈, 대시보드)
3. **`hooks/`** - 커스텀 훅. (예: API 호출, 인증 로직)
4. **`context/`** - 전역 상태 관리. (예: 사용자 인증, 다크 모드)
5. **`services/`** - API 호출 로직. (예: 백엔드와 통신)
6. **`utils/`** - 유틸리티 함수. (예: 날짜 포맷, 디바운스)
7. **`assets/`** - 이미지, 아이콘 등 정적 파일.
8. **`styles/`** - 전역 스타일, CSS 파일.

---

더 간단하게 요약하자면:
- **UI**는 `components/`, **화면**은 `pages/`, **로직**은 `hooks/`와 `services/`,  
- **공통 자원**은 `utils/`와 `assets/`.
